## Informations générales 

Ceci est un projet Fullstack fait en première année de l'école IT d'Orléans. Il vient valider notre Module de Web. Dans notre equipe on est 2, Bo-Br et 4h4ze : Bo-Br majoritairement backend, 4h4ze majoritairement frontend.

------

## Sujet

Ceci est un site vitrine d'un barbershop local venant d'Orleans. Il vise un très petit barbershop avec un seul coiffeur, et donc un seul admin. 

------

## Stack

Frontend : 
- Html
- Css
- Javascript

Backend : 
- PHP
- SQL

## Installation
#### Téléchargez (clonez) les fichiers de ce répository git.
Sur votre ordinateur :
1. Créer un dossier.
- Click droit dans votre navigateur de fichiers.
- Choisir "New" (ou "nouveau").
- Chosir "Folder" (ou "dossier").
- <img width="678" height="427" alt="image" src="https://github.com/user-attachments/assets/3c4e5379-205c-4655-900f-693a1c9f5cb8" />
- Un dossier doit apparaitre
- <img width="323" height="133" alt="image" src="https://github.com/user-attachments/assets/5503d6ca-65f7-4328-9fef-960ac0739e8d" />
2. Copier les fichiers
- Faites un double click sur votre dossier pour y acceder.
- Ouvrez une ligne de commande dans ce fichier.
- Click droit dans votre navigateur de fichiers.
- Choisir "Open in terminal" (ou "ouvrir dans le terminal").
- <img width="342" height="412" alt="image" src="https://github.com/user-attachments/assets/0464e073-9d8b-47b0-8d27-a1998dfe22bc" />


- tapez ```git clone https://github.com/Bo-Br/Fullstack_Web_Project``` et validez. Ceci doit copier tout les fichiers dans votre dossier.
